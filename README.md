# vue-library-tmp

> Vue 依赖包项目工程模版
>
> 相关博客: <https://bin.zmide.com/?p=1094>

## 开始 👾
```
npm install
```

### 启动 🚀
```
npm run serve
```

### 编译 🗜
```
npm run build
```

### 检查修复 🛴
```
npm run lint
```
